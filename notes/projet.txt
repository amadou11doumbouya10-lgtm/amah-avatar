----------------------------------------------------------------
  C'EST QUOI CE PROJET ?
----------------------------------------------------------------

Un avatar IA public qui représente la marque "The Amah" sur
le web et les réseaux sociaux. Il combine deux fonctions :

  MODE CHATBOT  → Les visiteurs lui parlent directement
  MODE AGENT    → Il enregistre, analyse et rapporte tout

Créé dans le cadre de l'écosystème Theamah :
  → Theamah+ (site streaming)
  → Theamah (marque tech)
  → TikTok cyber content

----------------------------------------------------------------
  FICHIER : index.html
----------------------------------------------------------------

C'est la page principale que voient les visiteurs.

FONCTIONNALITES :
  - Chatbot avec la persona "The Amah"
  - Design sombre/or, typographie élégante (Cormorant + DM Mono)
  - Effet typewriter sur les réponses de l'IA
  - 5 boutons de démarrage rapide (starter chips)
  - Compteur de messages : 0 à 20 par session
  - Alerte à 16/20, blocage à 20/20
  - Bouton RESET pour nouvelle conversation
  - Toast de notification
  - Responsive mobile

PERSONA DE L'AMAH (dans le script) :
  - Parle de streaming, tech, IA, cybersécurité, lifestyle
  - Redirige vers Theamah+ pour les films/séries
  - Mentionne Theamah comme créateur
  - Style : court, dense, élégant, mystérieux
  - Jamais de listes, jamais de "Bonjour !"
  - Max 4 phrases par réponse

MODELE IA UTILISE :
  - claude-sonnet-4-6 (Anthropic)
  - Max 400 tokens par réponse
  - Fenêtre historique : 10 derniers échanges

COMMENT ACTIVER LE CHAT :
  Option A (privé) : mettre ta clé dans CONFIG.apiKey
  Option B (public) : déployer sur Netlify + remplir CONFIG.apiEndpoint

LOGGING AUTOMATIQUE (côté agent) :
  - Chaque échange est sauvegardé dans le navigateur
  - Clé localStorage : "amah_logs"
  - Format : { id, sessionId, date, user, ai }
  - Limite : 1000 entrées max
  - Visible dans admin.html

----------------------------------------------------------------
  FICHIER : netlify/functions/chat.js
----------------------------------------------------------------

Ce fichier permet de rendre le chatbot 100% public sans que
les visiteurs aient besoin d'une clé API.

COMMENT CA MARCHE :
  1. Tu déploies le projet sur Netlify (gratuit)
  2. Tu ajoutes ta clé API dans les paramètres Netlify
  3. Les visiteurs arrivent sur index.html et chattent
     directement — sans aucune clé requise de leur part

PROTECTION INCLUSE :
  - Rate limiting : 30 messages max par IP par heure
  - La clé API ne jamais visible dans le code public
  - Headers de sécurité (CORS, X-Frame-Options)

VARIABLE D'ENVIRONNEMENT A CONFIGURER :
  Nom     : ANTHROPIC_API_KEY
  Valeur  : sk-ant-... (ta vraie clé)


----------------------------------------------------------------
  FICHIER : netlify.toml
----------------------------------------------------------------

Fichier de configuration pour le déploiement sur Netlify.
  - Indique où sont les fonctions (netlify/functions/)
  - Ajoute des headers de sécurité sur tout le site
  - Cache admin.html des moteurs de recherche (noindex)


----------------------------------------------------------------
  ETAPES POUR METTRE EN LIGNE (NETLIFY)
----------------------------------------------------------------

  ETAPE 1 — Préparer le dossier
    → S'assurer que tous les fichiers sont dans "avatar Amah/"

  ETAPE 2 — Créer un compte Netlify
    → https://netlify.com (gratuit)
    → Se connecter avec GitHub ou email

  ETAPE 3 — Déployer
    → Netlify → "Add new site" → "Deploy manually"
    → Glisser-déposer le dossier "avatar Amah"

  ETAPE 4 — Ajouter la clé API
    → Netlify → ton site → Site settings
    → Environment variables → Add variable
    → Nom : ANTHROPIC_API_KEY
    → Valeur : ta clé sk-ant-...

  ETAPE 5 — Activer la fonction dans index.html
    → Ouvrir index.html
    → Chercher : CONFIG.apiEndpoint = ''
    → Remplacer par : CONFIG.apiEndpoint = '/.netlify/functions/chat'

  ETAPE 6 — Redéployer
    → Re-glisser le dossier sur Netlify

  ETAPE 7 — Tester
    → Aller sur l'URL Netlify générée
    → Le chat doit s'ouvrir directement sans demander de clé
